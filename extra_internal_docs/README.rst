============================
Documentos internos anuldos
============================

Permite el registro de documentos anulados no relacionados con el servicio de rentas internas.

Se usa principalmente para control interno de la empresa.

Installation
============

To install this module, you need to:

* Add the module to your addons path.
* Install the module as usual.
